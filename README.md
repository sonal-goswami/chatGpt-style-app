# ChatGPT Style Application

This contains a simple working fullstack App:
- Backend: Node.js + Express (JavaScript). Mock JSON files for sessions and responses.
- Frontend: Vite + React + TypeScript + Tailwind. Simple UI with sessions, question input, table responses, like/dislike.

## Run locally

### Backend
- cd backend
- npm install
- npm run dev
- Server: http://localhost:3000

### Frontend
- cd frontend
- npm install
- npm run dev
- Open: http://localhost:5173
